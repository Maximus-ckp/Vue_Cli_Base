# Vue-Base-Project

本项目基于Vue-cli 3.x ，用于基本项目启用使用

## Project setup
```shell
npm install
```

### Compiles and hot-reloads for development
```shell
npm run serve / npm run start / npm run dev
```

### Compiles and minifies for production
```shell
# for UAT environment
npm run build-uat
# for Production environment
npm run build-prod
```

### Run your tests
```shell
npm run test
```

### Lints and fixes files
```shell
npm run lint
```
